About the visualizations

Scatterplot

The scatter plot is a uniform random sample of all data-points in the given timeframe with the given sample-size.
Zooming is possible via click-hold selection. A double click resets the time axis and custom sample-size.

Lineplot

Barchart

Boxplot

The box plot is a data distribution visualization of all data-points in the given timeframe.

Bubblechart

Chordgraph

Correlation Analysis of daily data

This visualization contains a correlation analysis of the data aggregated per day. In this analysis, the sum of calories and steps is determined, and the mean value is determined for temperature and heart rate. The correlation measure (Pearson correlation coefficient) is automatically displayed. Regression equations can be inserted through the data points and linear, quadratic and polynomial (order=3) regression equations can be displayed with the corresponding R2-value.
